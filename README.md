# What is this?
We are benchmarking the ability for different models to give correct answers to Agronomy questions. The benchmark is a set of multiple-choice questions spanning several categories — nutrient management, crop management, pest management, soil & water, plus community-contributed and V1 questions — and now also includes a vision-based **Image Identification** category. Models answer with a single answer key, which is scored against the correct answer.

The questions and images themselves are intentionally kept out of this repository so the benchmark can't be gamed; only the aggregate leaderboard and charts are published here.

# Image Identification (vision models)
This category tests whether vision-capable models can identify agronomic subjects from a single photo. Each question shows one image and asks the model to pick the correct label from four options, covering three domains:

- **Crop disease** — leaf images labeled by plant and condition (source: PlantVillage)
- **Insect pests** — labeled pest photos (source: IP102)
- **Weeds** — labeled weed species (sources: DeepWeeds and CottonWeedID15)

Details:
- **Coverage:** 150 questions (50 disease, 50 pests, 50 weeds).
- **Vision gating:** image questions are only sent to models that accept image input. Text-only models are shown as `N/A` rather than scored as wrong.
- **Scoring:** each image domain is reported as its own column (`Image Disease`, `Image Pest`, `Image Weed`) and all are **excluded from the Overall Score**, so the text-based overall scores remain directly comparable across every model.
- **Privacy:** the images and questions are not committed to this repository.

# Why?
When building new models for agriculture, it's important to know if your model is getting better or worse. This is a simple benchmark to help us determine if we are improving the agronomic ability of new models and by how much.

# Roadmap
1. Make it harder! These are fairly basic questions. We should add short and long answer questions (to be evaluated against example correct answers)
2. Add questions for international regions
3. Add more models to the leaderboard
